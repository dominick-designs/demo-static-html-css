How you can create a web page raw HTML

HTML

HTML Centers around text.

Think of it like a word processor. It modifies, adjusts, and refines text.

CSS

CSS Centers around modifying the appearance of text.

Think of it like a photo editor. You can add colors, space, and even animation.

JavaScript

JavaScript Centers around applying action to elements.

Think of it like an oven. This is where you make something new from raw ingredients.
